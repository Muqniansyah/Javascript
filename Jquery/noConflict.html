Metode jQuery noConflict()

Seperti yang sudah Anda ketahui; jQuery menggunakan $tanda sebagai jalan pintas untuk jQuery. Ada banyak framework JavaScript populer lainnya seperti: Angular, Backbone, Ember, Knockout, dan lainnya. Bagaimana jika framework JavaScript lain juga menggunakan tanda $ sebagai jalan pintas? Jika dua kerangka kerja berbeda menggunakan pintasan yang sama, salah satunya mungkin berhenti bekerja. Tim jQuery sudah memikirkan hal ini, dan mengimplementasikan metodenya noConflict().

Metode noConflict()melepaskan penangguhan pada pengenal pintasan $, sehingga skrip lain dapat menggunakannya. Anda tentu saja masih bisa menggunakan jQuery, cukup dengan menuliskan nama lengkapnya alih-alih pintasannya:

This is a paragraph.
