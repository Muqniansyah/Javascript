x-on:{nama_event} / @{nama_event} untuk mendengarkan event

event yang bisa digunakan yaitu semua event yang ada dijavascript.


$event untuk event object

kita bisa mengakses sebuah magic property punya alpine yaitu $event. $event berguna untuk jika kita butuh informasi atau butuh object dari element yang sedang kita berikan event.

jadi kita bisa tahu element apa yang sedang kita klik dan informasi apa saja yang ada didalamnya.


@keyup dan @keydown untuk keyboard events

event @keyup ketika terlepas dari tombol pada keyboard. dan event @keydown ketika menekan tombol pada keyboard.

biasanya yang sering digunakan yaitu @keyup. karna untuk @keydown jika menekan terlalu lama maka javascript akan memanggil fungsinya secara terus menerus. namun jika @keyup maka javascript akan memanggil fungsinya sekali pada saat kita melepas tombol dari keyboard.
