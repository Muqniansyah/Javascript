x-model untuk two way data binding

model ini adalah sebuah cara agar kita dapat mengubah data yang sudah kita bikin dicomponent kita menggunakan element element input dari user. bisa menggunakan text-field, combobox, checkbox, dan lain lain.

untuk directives x-model data yang bisa merubah yaitu user dengan menggunakan element element input. macam macam element inputnya yaitu : input text, textarea, checkbox, radio button, select.
