untuk bisa menjalankan code dijavascript ada beberapa cara:
1.dengan search engine, seperti google atau mozilla
2.dengan npm dan node js sebagai package dari javascript
3.dengan menggunakan extension code runner pada vscode

disini node_modules tidak diinclude ke github maka untuk menjalankan folder latihan yang memiliki package(baik berupa framework, library, tools, plugin atau lainnya) pada packge.json, maka harus diinstal ulang pakage 
devDependencies dengan mengetikkan perintah "npm install" pada terminal berdasarkan foldernya yang ingin digunakan. tapi pastikan sudah mendownload npm maupun node js.
